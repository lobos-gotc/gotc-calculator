# Data Extraction Tools

The data extraction tools for Game of Thrones: Conquest have been moved to a separate branch for better organization.

## Location

All extraction tools, scripts, and extracted data are available in the **`data-extraction-tools`** branch:

```bash
git checkout data-extraction-tools
```

## What's Included

- **Protobuf Decoders** - Extract data from game's binary files
- **Screenshot Automation** - Automated hero screenshot collection
- **Data Entry Tools** - Assisted tools for manual data verification
- **Unity Sprite Extractors** - Extract images from Unity bundles
- **Complete Hero Database** - 114 heroes with council positions
- **Documentation** - Full guides for all extraction workflows

## Quick Access

View the tools on GitHub:
https://github.com/angcadena/crafting-tool/tree/data-extraction-tools

## Future Plans

These tools will be moved to a separate repository for:
- Easier maintenance
- Reusability across other projects
- Independent versioning
- Better documentation organization

## For Developers

If you need to extract new game data or update existing data, switch to the `data-extraction-tools` branch and refer to the documentation there.
